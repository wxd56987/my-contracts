# Advertisements Project

## Install Dependencies

```sh
yarn
```

## Command

Compile contracts

```sh
yarn compile
```

Test contracts
```sh
yarn test # Run all test case
yarn test test/Advertisements.spec.ts # Run single case
```

Clean caches
```sh
yarn clean
```
